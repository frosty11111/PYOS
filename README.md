# PYOS
PYOS is a game-like app that (tries) to imitate a normal Operating System made purely on python.

Version deciphering:
the current version of PYOS is 0.2.0 ALPHA TEST
0. -- Major updates
2. -- Mild updates (built-in software and or gui fixes)
0 -- Small updates (bug-fixes, patches, etc.)
ALPHA -- Current stage of testing

Terminal usage:
Terminal is a built-in app what will open when you click on "Terminal" button on the taskbar
Currently there are 8 commands:
1. help -- outputs a list of all available commands
2. calc/calculator -- opens built-in calculator (in the terminal)
3. clear -- clears the terminal
4. exit -- aborts all running PYOS windows (gui, ide, terminal)
5. timer -- sets up a timer that will block any input in terminal it was started (input time in seconds)
6. text editor -- opens up notepad
7. browser -- opens up newly-made browser
8. crash -- forces PYOS into ESoD (Error Screen of Death)
